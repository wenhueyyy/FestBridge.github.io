# 中国传统节日小程序项目文档

## 项目概述

这是一个专注于中国传统节日的信息展示和智能助手项目，提供节日日历、详细信息、智能聊天等功能，支持中英文切换。

## 项目结构

```
├── index.html          # 主页
├── calendar.html       # 节日日历
├── chat.html           # 智能聊天助手
├── detail.html         # 节日详情页
├── blessings.html      # 祝福语库
├── customs.html        # 节日习俗
├── items.html          # 必备物品
├── tips.html           # 文化避坑指南
├── upcoming.html       # 即将到来的节日
├── profile.html        # 个人中心
├── about.html          # 关于页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── app.js          # 核心功能
│   ├── api.js          # Coze API调用
│   └── lang.json       # 多语言支持
└── images/             # 图片资源
```

## 核心功能

### 1. 节日日历

- **功能**：展示公历和农历节日
- **特点**：
  - 点击节日自动跳转到详情页
  - 支持中英文切换
  - 格子大小一致，排版美观
  - 点击日期后在下方显示该日期的节日信息
  - 节日栏中的节日名称可点击跳转详情页

### 2. 智能聊天助手

- **功能**：提供节日相关咨询
- **技术实现**：
  - 调用Coze API实现智能对话
  - 支持流式响应（SSE）
  - 实时更新显示
  - 建议问题功能
  - 英文空格修复
  - 防重复发送机制
- **特点**：
  - 多轮对话
  - 智能回答节日相关问题
  - 支持中英文回复
  - 显示建议问题

### 3. 多语言支持

- **功能**：实现整体中英文顺畅切换
- **技术实现**：
  - 使用lang.json存储多语言文本
  - 动态更新页面元素
  - 本地存储用户语言偏好
- **特点**：
  - 支持完全的中英文切换
  - 所有页面元素都能正确翻译
  - 节日名称和描述的双语支持

### 4. 节日详情

- **功能**：展示节日的详细信息
- **内容**：
  - 节日介绍
  - 历史起源
  - 传统习俗
  - 祝福语
  - 相关物品
  - 文化注意事项

### 5. 其他功能

- **祝福语库**：提供各种节日的祝福语
- **习俗指南**：详细介绍节日习俗
- **必备物品**：列出节日所需物品
- **文化避坑指南**：提供文化注意事项
- **即将到来**：展示即将到来的节日
- **个人中心**：用户个人信息和设置

## 技术实现

### 前端技术

- **HTML5**：页面结构
- **CSS3**：样式设计，包括响应式布局
- **JavaScript**：交互逻辑和功能实现
- **LocalStorage**：存储用户偏好和状态

### API集成

- **Coze API**：实现智能聊天功能
- **SSE (Server-Sent Events)**：处理流式响应
- **Fetch API**：网络请求

### 关键技术点

1. **流式响应处理**：
   - 使用`ReadableStream`和`TextDecoder`处理SSE
   - 实时更新聊天界面
   - 防重复内容处理

2. **多语言支持**：
   - 动态切换语言
   - 本地存储语言偏好
   - 自动更新页面元素

3. **节日日历**：
   - 公历和农历日期处理
   - 节日数据管理
   - 点击事件和页面跳转

4. **英文空格修复**：
   - 智能修复英文单词之间的空格
   - 常见节日名称和错误组合的修复
